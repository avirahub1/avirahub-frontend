# **App Name**: AviralHub

## Core Features:

- Interactive 3D Hero: A fullscreen hero section that reacts to user interactions, creating a dynamic 3D experience.
- Scroll-Based Narrative Transitions: Smooth transitions between sections triggered by scroll position for a seamless single-page experience.
- Service Pedestal Interactions: Interactive pedestals showcasing core services with engaging hover effects.
- 3D Project Gallery with Rail Effect: A gallery with floating 3D-style screens for portfolio projects, featuring a camera rail effect on scroll.
- Tactical 'Why Us' HUD: A minimal, HUD-style UI that presents key reasons to choose AviralHub.
- Contact Form Terminal: A contact form presented as a floating 'terminal' with input validation and feedback messages.
- AI Content Generator: Utilize a generative AI tool to assist users with drafting initial project details within the contact form, ensuring clarity and relevance. The LLM will use reasoning.

## Style Guidelines:

- Primary color: Dark grey (#1F2933) for a sophisticated and muted background.
- Background color: Almost black (#050509), desaturated and dark to emphasize content.
- Accent color: Soft light gray (#D4D4D8) for borders, highlights, and buttons to create subtle contrast.
- Font: 'Inter' sans-serif for both headings and body text. Note: currently only Google Fonts are supported.
- Minimal, monochrome 3D-style icons that support the text without overpowering it. Simple geometric shapes and matte shading.
- Fullscreen environments with multi-layer parallax to create depth and a 3D feeling.
- Smooth, Unreal Engine-inspired animations for camera-like movements, transitions, and micro-interactions.
